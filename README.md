# Text-Preprocessing-using-NLTK
A Streamlit-based Text Preprocessing Tool that lets users perform Tokenization, Stopword Removal, Stemming, and Lemmatization interactively. Built with NLTK and Python, it features a clean pastel UI, gradient buttons, and real-time text processing for NLP beginners and learners.
